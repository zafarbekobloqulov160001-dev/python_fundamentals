# Python Dictionaries - Contact Manager

This project demonstrates how dictionaries work in Python.

## Concepts Covered

- Creating dictionaries
- Adding key-value pairs
- Searching in dictionaries
- Deleting dictionary items
- Iterating through dictionaries

## Features

- Add contacts
- View contacts
- Search contacts
- Delete contacts

## Example

```python
contacts = {

    "Bob": "987654321"
}
