# Python Sets - Unique Word Analyzer

This project demonstrates how sets work in Python.

## Concepts Covered

- Creating sets
- Removing duplicate values
- Converting lists to sets
- Iterating through sets

## Features

- Finds unique words in a sentence
- Counts total and unique words
- Demonstrates duplicate removal

## Example

```python
words = ["apple", "banana", "apple", "orange"]
unique = set(words)

print(unique)
