# Python Lists - To-Do List Manager

This project demonstrates how Python lists can be used to build
a simple To-Do List Manager.

## Concepts Covered

- Creating lists
- Adding items to a list
- Removing items from a list
- Iterating through lists using loops

## Features

- View tasks
- Add new tasks
- Remove tasks
- Exit the program

## Example

```python
tasks = []
tasks.append("Study Python")
tasks.append("Practice Git")
