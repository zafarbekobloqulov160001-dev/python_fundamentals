# Password Strength Checker

This is a simple Python project that checks how strong a password is.

The program analyzes the password based on several conditions and classifies it as **Weak**, **Medium**, or **Strong**.

## Features

- Checks password length
- Detects uppercase letters
- Detects numbers
- Detects special characters
- Classifies password strength

## Technologies

- Python
- Basic loops
- Conditional statements
